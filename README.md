# COGNIFYZ_L2_Task2

**💰 Task 02 - Price Range Analysis**

**🎯 Objective**

**To analyze restaurant price ranges and study their relationship with aggregate ratings and rating colors.**

**🛠️ Tools Used**

**    • Python
    • Pandas 
    • Matplotlib
    • Seaborn
    • Google Colab**

**📌 Steps Performed**

****1. Loaded dataset.

  2. Calculated average ratings for each price range
  
  3. Explored Prize Range Distribution.
  
  4. Identified rating colors associated with price ranges
   
  5. Identified the most common price range
  
  6. Determined the color representing the highest average rating****


  **💡 Key Insights**

  **Certain price ranges are more common among restaurants
  
  Average ratings vary across different price categories
  
  Higher price ranges tend to have better ratings in some cases
  
  Rating colors help visually represent restaurant quality
